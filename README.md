# wordle
this is wordle made in c++ but it's only for programming terms.
I made it because I was beginning to forget a lot of C++, and this side project that was a lot of fun. It was a great memory refresher.
# Rules
All the rules are the same as regular Wordle. This tests 5-letter string words, except it is random every day and rotates between a list of programming words to help you discover new terms. New words were last added **4/15/26**. Also, it doesn't reject nonsense words like Wordle does. If you don't know what to guess, just write a random string of letters and hope for the best. (The truth is I forgot to implement that but let's call it a feature).
